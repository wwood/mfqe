```
Usage: mfqe --fastq <fastq> --fastq-read-name-lists <LIST1> .. --output-fastq-files <OUTPUT1> ..

Purpose is to extract one or more sets of reads from a FASTQ file by specifying their read name.

Read name files are uncompressed text files with read names (without comments).
Output is gzip-compressed, input may or may not be.
```
